Sega System SP

The Sega System SP was released around the same time as the Sega Lindbergh, and is ultimately a more efficent copy of the Sega Naomi (without the white case)! Onboard it has a WindRiver ethernet controller for netbooting, an onboard I/O port for connecting controllers and the same processors and chips as the Naomi 1 for the most part. Games normally run off of an SD Card, however the board also has some internal memory which games such as Love and Berry can run off entierly.

Netbooting

I have managed to confirm the Netbooting functionality on this board, however have never managed to successfully boot a game. The netbooting seems to complete too quickly, and the game just gets stuck waiting for more data - I'm assuming that this board may use a slightly different Netbooting protocol?
